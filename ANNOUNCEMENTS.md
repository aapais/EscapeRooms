# Anúncios (copy‑paste)

## Mensagem inicial (Teams/Slack)
Pessoal, vamos fazer um mini “escape room” de engenharia com prémios.

1) Criem/juntem-se a uma equipa (2–4 pessoas)
2) Aceitem o assignment aqui: <LINK_GITHUB_CLASSROOM>
3) Abram o repo no VS Code e corram:
- `npm install`
- `npm run score`

O objetivo é desbloquear as rooms (tests/lint/scan) e maximizar o score.

## 10 min antes do fim
Faltam 10 minutos! Façam push para correr o autograde.
Se puderem, abram um PR chamado `final-score` para vermos o score no comentário.

## Final
Tempo! Vamos ver o leaderboard e anunciar vencedores.
