# Quick Start (para equipas)

## Requisitos
- Node.js 18+

## Começar
Na raiz do repo:
```bash
npm install
npm run score
```

## Gates por sala
- Room 1: `npm run room1`
- Room 2: `npm run room2`
- Room 3: `npm -w @escape/room3 run scan`
- Final: `npm run final`

## Entrega / score
- Faz push para correr GitHub Actions
- (Opcional) cria PR `final-score` para ter comentário automático com o TOTAL
